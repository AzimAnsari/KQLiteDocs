KQLite Columns

KQLite not only supports standard SQLite data types but also provides a rich set of column types designed to seamlessly work with native Kotlin data types.

SQLite standard columns

object StandardSQLiteTypesTable : KQLiteTable("standard_types_table") {
    val integerColumn: KQLiteColumn<Long?> = integerColumn("integer_column") // INTEGER

    val realColumn: KQLiteColumn<Double?> = realColumn("real_column") // REAL

    val textColumn: KQLiteColumn<String?> = textColumn("text_column") // TEXT

    val blobColumn: KQLiteColumn<ByteArray?> = blobColumn("blob_column") // BLOB

    val booleanColumn: KQLiteColumn<Boolean?> = booleanColumn("boolean_column") // BOOLEAN

    val notNullColumn: KQLiteColumn<String> = textColumn("not_null_column").notNull() // TEXT NOT NULL
}
