KQLite/com.kqlite.column/OrderColumn

OrderColumn

class OrderColumn @JvmOverloads constructor(column: KQLiteColumn<*>, collate: Collate? = null, sort: Sort? = null, nulls: Nulls? = null)

Represents a column used in an ORDER BY clause, specifying the sorting criteria.

This class encapsulates a database column along with optional sorting parameters such as collation sequence, sort order (ascending/descending), and nulls ordering.

Example

Employees
        .select()
        .orderBy(
            OrderColumn(
                column = Employees.firstName,
                collate = Collate.NOCASE,
                sort = Sort.DESC,
                nulls = Nulls.LAST,
            ),
            OrderColumn(
                column = Employees.lastName,
                collate = Collate.NOCASE,
                sort = Sort.ASC,
                nulls = Nulls.LAST,
            ),
        ).execute()
