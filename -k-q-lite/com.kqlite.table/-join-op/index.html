JoinOp

enum JoinOp : Enum<JoinOp>

Represents the comparison operators used in a JOIN clause in SELECT statement.

If not specified, default operator is JoinOp.EQ (=)

Fore usage see: com.kqlite.statement.JoinStatement.on

Supported operators are:

  • JoinOp.EQ (=)

  • JoinOp.NOT_EQ (!=)

  • JoinOp.GT (>)

  • JoinOp.GTE (>=)

  • JoinOp.LT (<)

  • JoinOp.LTE (<=)
